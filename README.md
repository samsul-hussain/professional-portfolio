# Professional Portfolio

This repository contains my personal and professional portfolio website, designed to showcase my skills, services, and experience as a Digital Marketer. 

## Features
- **Responsive Design**: Fully responsive layout that looks great on mobile, tablet, and desktop devices.
- **Modern UI/UX**: Clean, professional aesthetic using modern design principles.
- **Interactive Elements**: Dynamic micro-animations and smooth scrolling for an engaging user experience.
- **SEO Optimized**: Built with semantic HTML and structured for search engine visibility.

## Tech Stack
- HTML5
- CSS3 (Vanilla CSS, custom properties, flexbox/grid)
- JavaScript (Vanilla)

## Setup & Local Development
1. Clone this repository to your local machine.
2. Open `index.html` in your web browser of choice.
3. No build tools or package managers are required.
